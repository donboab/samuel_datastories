Project Description

This project demonstrates the use of structured query language (SQL) to analyze and extract actionable insights from a relational database containing employee records. The objective was to apply real-world SQL techniques to solve common business and HR-related queries, including workforce segmentation, salary analysis, and data transformation. The analytical focus areas are:

Data Structure & Tables:

The analysis was based on two relational tables; (Employee_Info and Employee_Position). These tables contains employee demographic details, departmental affiliations, job roles, compensation data etc.

Query Objective Areas:

  • Data formatting and transformation (e.g., name casing, string parsing).
  • Filtering and pattern matching (e.g., employees with names starting or ending with specific letters).
  • Date manipulation and temporal analysis (e.g., filtering by date of birth or date of joining).
  • Aggregation and grouping (e.g., department-wise employee count, gender-based segmentation).
  • Salary and financial insights (e.g., range filters, top earners, min/max salary brackets).
  • Table creation and joins for enriched analysis across tables.

Impact of the Project

  • Developed over 20 reusable SQL queries to support workforce and salary analysis.
  • Simulated real HR data tasks like employee segmentation, compensation analysis, and department summaries.
  • Delivered insights that can inform staffing decisions, compensation reviews, and talent analytics.
